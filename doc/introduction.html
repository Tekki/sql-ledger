Introduction

Quick Start

To establish a fully operational system, follow these steps:

  • Install the server: Installation.
  • Configure the system: Global Configuration.
  • Create at least one database.
  • Log in to this database.
  • Edit the preferences of the administrator.
  • Edit the chart of accounts via System → Chart of Accounts.
  • Configure the database: Database Configuration.
  • Add at least one user without admin permissions via the HR menu: User Configuration.
